AC-Method Proof Using Substitution 用代换法证明 AC 方法

A clean proof that explains why the AC-method works, followed by a shorter version and a full example. 下面给出一个较清楚的证明，说明为什么 AC 方法成立，并附上简短版证明和完整例题。

Proof of the AC-Method Using Substitution 用代换法证明 AC 方法

Step 1 第 1 步

Multiply by a 先乘上 a

Start with 从

ax2 + bx + c

We want to factor it. Multiply the expression by a: 我们想把它因式分解。先把整个式子乘上 a：

a(ax2 + bx + c) = a2x2 + abx + ac

Now let 现在设

y = ax

Then the right-hand side becomes 那么右边就变成

y2 + by + ac

So instead of factoring ax2 + bx + c, we first factor the monic quadratic y2 + by + ac. 所以，与其直接分解 ax2 + bx + c，我们先去分解首项系数为 1 的二次式 y2 + by + ac。

Step 2 第 2 步

Factor the monic quadratic 分解首项系数为 1 的二次式

Suppose we can find numbers m and n such that 假设我们能找到两个数 m 和 n，满足

m + n = b,     mn = ac

Then 那么

y2 + by + ac = (y + m)(y + n)

because 因为

(y + m)(y + n) = y2 + (m + n)y + mn = y2 + by + ac
Step 3 第 3 步

Substitute back y = ax 把 y = ax 代回去

Since y = ax, we get 由于 y = ax，可得

a(ax2 + bx + c) = (ax + m)(ax + n)

So 所以

ax2 + bx + c = [(ax + m)(ax + n)] / a

This is the key identity behind the AC-method. 这就是 AC 方法背后的关键恒等式。

Step 4 第 4 步

Why dividing by a gives the correct linear factors 为什么除以 a 之后会得到正确的一次因式

We now justify how the factor a is absorbed into the two binomials. 现在说明为什么这个因子 a 可以分配到两个二项式中。

Let d = gcd(a, m). Write 设 d = gcd(a, m)。写成

a = de,     m = dr

with gcd(r, e) = 1. Since mn = ac = dec, we have 其中 gcd(r, e) = 1。由于 mn = ac = dec，所以有

drn = dec

Cancel d: 约去 d：

rn = ec

Because gcd(r, e) = 1, Euclid's lemma shows that e divides n. So write 因为 gcd(r, e) = 1，由欧几里得引理可知 e 整除 n。因此可写成

n = es

Then rs = c, and 于是 rs = c，并且

(ax + m)(ax + n) = (dex + dr)(dex + es)

Factor each bracket: 对每个括号提公因式：

(dex + dr)(dex + es) = de(ex + r)(dx + s) = a(dx + s)(ex + r)

Substitute this into the identity from Step 3: 把它代回第 3 步得到的恒等式：

a(ax2 + bx + c) = a(dx + s)(ex + r)

Divide by a: 再除以 a：

ax2 + bx + c = (dx + s)(ex + r)

Also, from m = dr and n = es, we get 另外，由 m = dr 和 n = es，可得

b = m + n = dr + es

and from rs = c with de = a, the expansion of (dx + s)(ex + r) is exactly 并且由 rs = c 与 de = a，展开 (dx + s)(ex + r) 正好得到

dex2 + (dr + es)x + rs = ax2 + bx + c

Thus the AC-method produces the correct factorization. 因此，AC 方法确实给出了正确的因式分解。

Shorter Version of the Proof 简短版证明

Given 给定

ax2 + bx + c

multiply by a: 先乘以 a：

a2x2 + abx + ac

Let y = ax. Then we get 设 y = ax。则得到

y2 + by + ac

If m + n = b and mn = ac, then 如果 m + n = b 且 mn = ac，那么

y2 + by + ac = (y + m)(y + n)

Substitute back y = ax: 把 y = ax 代回去：

a(ax2 + bx + c) = (ax + m)(ax + n)

So 所以

ax2 + bx + c = [(ax + m)(ax + n)] / a

Since mn = ac, the factor a can be split between m and n, which simplifies the result into two linear factors. That is why the AC-method works. 由于 mn = ac，因子 a 可以分配到 m 和 n 中，从而化成两个一次因式。这就是 AC 方法成立的原因。

Example in This Proof Style 按这种证明风格写的例子

Factor 分解

12x2 - 5x - 2

Multiply by 12: 先乘以 12：

144x2 - 60x - 24

Let 设

y = 12x

Then we get 则得到

y2 - 5y - 24

Factor: 因式分解：

y2 - 5y - 24 = (y - 8)(y + 3)

Substitute back: 代回去：

12(12x2 - 5x - 2) = (12x - 8)(12x + 3)

So 所以

12x2 - 5x - 2 = [(12x - 8)(12x + 3)] / 12

Now simplify: 现在化简：

[(12x - 8)(12x + 3)] / 12 = [4(3x - 2) x 3(4x + 1)] / 12 = (3x - 2)(4x + 1)
Final factorization: 最终分解结果：
12x2 - 5x - 2 = (3x - 2)(4x + 1)

Final Polished Proof Statement 最终整理后的证明结论

For any quadratic ax2 + bx + c, multiply by a and let y = ax. Then 对于任意二次式 ax2 + bx + c，先乘以 a，再设 y = ax。则

a(ax2 + bx + c) = y2 + by + ac

If 如果

y2 + by + ac = (y + m)(y + n)

where m + n = b and mn = ac, then 其中 m + n = b 且 mn = ac，那么

a(ax2 + bx + c) = (ax + m)(ax + n)

Dividing by a and simplifying produces the factorization of ax2 + bx + c. Therefore the AC-method is valid. 再除以 a 并化简，就得到 ax2 + bx + c 的因式分解。因此，AC 方法是成立的。

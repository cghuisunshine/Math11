AC-Method Proof Using Substitution

A clean proof that explains why the AC-method works, followed by a shorter version and a full example.

Proof of the AC-Method Using Substitution

Step 1

Multiply by a

Start with

ax2 + bx + c

We want to factor it. Multiply the expression by a:

a(ax2 + bx + c) = a2x2 + abx + ac

Now let

y = ax

Then the right-hand side becomes

y2 + by + ac

So instead of factoring ax2 + bx + c, we first factor the monic quadratic y2 + by + ac.

Step 2

Factor the monic quadratic

Suppose we can find numbers m and n such that

m + n = b,     mn = ac

Then

y2 + by + ac = (y + m)(y + n)

because

(y + m)(y + n) = y2 + (m + n)y + mn = y2 + by + ac
Step 3

Substitute back y = ax

Since y = ax, we get

a(ax2 + bx + c) = (ax + m)(ax + n)

So

ax2 + bx + c = [(ax + m)(ax + n)] / a

This is the key identity behind the AC-method.

Step 4

Why dividing by a gives the correct linear factors

We now justify how the factor a is absorbed into the two binomials.

Let d = gcd(a, m). Write

a = de,     m = dr

with gcd(r, e) = 1. Since mn = ac = dec, we have

drn = dec

Cancel d:

rn = ec

Because gcd(r, e) = 1, Euclid's lemma shows that e divides n. So write

n = es

Then rs = c, and

(ax + m)(ax + n) = (dex + dr)(dex + es)

Factor each bracket:

(dex + dr)(dex + es) = de(ex + r)(dx + s) = a(dx + s)(ex + r)

Substitute this into the identity from Step 3:

a(ax2 + bx + c) = a(dx + s)(ex + r)

Divide by a:

ax2 + bx + c = (dx + s)(ex + r)

Also, from m = dr and n = es, we get

b = m + n = dr + es

and from rs = c with de = a, the expansion of (dx + s)(ex + r) is exactly

dex2 + (dr + es)x + rs = ax2 + bx + c

Thus the AC-method produces the correct factorization.

Shorter Version of the Proof

Given

ax2 + bx + c

multiply by a:

a2x2 + abx + ac

Let y = ax. Then we get

y2 + by + ac

If m + n = b and mn = ac, then

y2 + by + ac = (y + m)(y + n)

Substitute back y = ax:

a(ax2 + bx + c) = (ax + m)(ax + n)

So

ax2 + bx + c = [(ax + m)(ax + n)] / a

Since mn = ac, the factor a can be split between m and n, which simplifies the result into two linear factors. That is why the AC-method works.

Example in This Proof Style

Factor

12x2 - 5x - 2

Multiply by 12:

144x2 - 60x - 24

Let

y = 12x

Then we get

y2 - 5y - 24

Factor:

y2 - 5y - 24 = (y - 8)(y + 3)

Substitute back:

12(12x2 - 5x - 2) = (12x - 8)(12x + 3)

So

12x2 - 5x - 2 = [(12x - 8)(12x + 3)] / 12

Now simplify:

[(12x - 8)(12x + 3)] / 12 = [4(3x - 2) x 3(4x + 1)] / 12 = (3x - 2)(4x + 1)
Final factorization:
12x2 - 5x - 2 = (3x - 2)(4x + 1)

Final Polished Proof Statement

For any quadratic ax2 + bx + c, multiply by a and let y = ax. Then

a(ax2 + bx + c) = y2 + by + ac

If

y2 + by + ac = (y + m)(y + n)

where m + n = b and mn = ac, then

a(ax2 + bx + c) = (ax + m)(ax + n)

Dividing by a and simplifying produces the factorization of ax2 + bx + c. Therefore the AC-method is valid.

Important Correction

  • It should be a*x^2 + bx + c, not a*x^2 + bx + cx.
  • After letting y = ax, the expression becomes y^2 + by + ac.
  • This follows exactly from a(ax^2 + bx + c) = a^2x^2 + abx + ac = (ax)^2 + b(ax) + ac.

1. Angles and Their Measures

Concepts
  • Initial Side: The starting ray of an angle. In standard position, the initial side lies along the positive x-axis.
  • Terminal Side: The ending ray of an angle after rotation from the initial side.
  • Standard Position: An angle is in standard position when its vertex is at the origin and its initial side lies along the positive x-axis.
  • Angle Rotation: The measure of an angle is determined by the amount and direction of rotation from the initial to the terminal side. A counterclockwise rotation results in a positive angle, while a clockwise rotation yields a negative angle.
  • Degree Measure: A full rotation is 360 degrees (360°). One degree (1°) is 1/360 of a complete rotation.
  • Radian Measure: A radian is another unit for measuring angles, where one full rotation is 2π radians. This relationship is derived from the unit circle, where the circumference is 2π.
  • Coterminal Angles: These are angles in standard position that share the same initial and terminal sides.
  • Reference Angle: The acute angle formed between the terminal side of an angle and the x-axis.
Formulas
  • Finding Coterminal Angles: To find angles that are coterminal with an angle θ, add or subtract multiples of 360°. In radians, add or subtract multiples of 2π.
  • Reference Angle Rules: For an angle θ between 0° and 360°:
    • Quadrant I: reference angle = θ
    • Quadrant II: reference angle = 180° - θ
    • Quadrant III: reference angle = θ - 180°
    • Quadrant IV: reference angle = 360° - θ
  • Degree to Radian Conversion: Multiply the degree measure by π/180°.
  • Radian to Degree Conversion: Multiply the radian measure by 180°/π.
Examples
  • Finding Coterminal Angles: To find two positive and two negative angles coterminal with 465°:
    • 465° - 360° = 105° positive
    • 465° + 360° = 825° positive
    • 465° - 2 × 360° = -255° negative
    • 465° - 3 × 360° = -615° negative
  • Finding a Reference Angle: For 240°, the terminal side is in Quadrant III, so the reference angle is:
    240° - 180° = 60°
  • Converting Degrees to Radians: To convert 240° to radians:
    240° × (π/180°) = 4π/3 radians
  • Converting Radians to Degrees: To convert 7π/5 radians to degrees:
    (7π/5) × (180°/π) = 252°

2. Arc Length

Concept

The length of a circular arc s is directly proportional to the radius r and the central angle θ. For the formula to work, the angle must be measured in radians.

Formula
s = rθ
  • s = arc length
  • r = radius
  • θ = central angle in radians
Example

Determine the arc length of a circle with a radius of 5 cm and a central angle of 60°.

  1. First, convert the angle to radians:
    60° × (π/180°) = π/3 radians
  2. Use the arc length formula:
    s = rθ = 5 × (π/3) = 5π/3 ≈ 5.24 cm

3. Trigonometric Functions

Concepts
  • Coordinate Plane Definition: For an angle θ in standard position with a point P(x, y) on its terminal side, the six trigonometric functions are defined as ratios of x, y, and r, where r = √(x2 + y2).
  • Signs of Functions: The quadrant in which the terminal side of the angle lies determines whether the trigonometric functions are positive or negative. The value of r is always positive.
    • sinθ and cscθ are positive where y is positive: Quadrants I and II.
    • cosθ and secθ are positive where x is positive: Quadrants I and IV.
    • tanθ and cotθ are positive where x and y have the same sign: Quadrants I and III.
Formulas
sinθ = y/r
cosθ = x/r
tanθ = y/x
cscθ = r/y
secθ = r/x
cotθ = x/y
Example

Given the point (2, -1) on the terminal side of angle θ, find the value of all six trigonometric functions.

  1. First, find r:
    r = √((2)2 + (-1)2) = √5
  2. Use the formulas with x = 2, y = -1, and r = √5:
Function Value
sinθ-1/√5
cosθ2/√5
tanθ-1/2
cscθ-√5
secθ√5/2
cotθ-2

4. Graphing Trigonometric Functions

Concepts
  • Periodic Function: A function is periodic if it repeats its values at regular intervals. The length of the shortest repeating interval is the period.
  • Amplitude: For a periodic function, the amplitude is half the distance between its maximum and minimum values.
  • Phase Shift: This is the horizontal shift of the graph.
  • Vertical Displacement: This is the vertical shift of the graph.
Formula

For functions of the form:

f(x) = a sin(b(x - c)) + d
f(x) = a cos(b(x - c)) + d
  • Amplitude: |a|
  • Period: 2π/|b| for sine and cosine, or π/|b| for tangent
  • Phase Shift: c, shift right if c > 0, left if c < 0
  • Vertical Displacement: d, shift up if d > 0, down if d < 0
Example

Find the transformations for y = -3 cos((π/4)x + π/2) + 1.

  1. Rewrite the function to isolate the phase shift:
    y = -3 cos((π/4)(x + 2)) + 1
  2. Identify the parameters:
    a = -3, b = π/4, c = -2, d = 1
    • Amplitude: |a| = |-3| = 3
    • Period: 2π/b = 2π/(π/4) = 8
    • Phase Shift: c = -2, which is a shift left by 2 units
    • Vertical Displacement: d = 1, which is an upward shift of 1 unit

5. Trigonometric Identities

Concepts
  • Identity vs. Equation: An identity is an equation that is true for all allowable values of the variable, whereas a conditional equation is true only for specific values.
  • Verifying Identities: To prove an identity, you typically work with the more complicated side of the equation and use fundamental identities and algebraic rules to transform it until it is identical to the other side.
Formulas: Fundamental Identities

Reciprocal Identities

cscθ = 1/sinθ
secθ = 1/cosθ
cotθ = 1/tanθ

Quotient Identities

tanθ = sinθ/cosθ
cotθ = cosθ/sinθ

Pythagorean Identities

sin2θ + cos2θ = 1
1 + tan2θ = sec2θ
1 + cot2θ = csc2θ

Sum and Difference Identities

cos(A ± B) = cosAcosB ∓ sinAsinB
sin(A ± B) = sinAcosB ± cosAsinB

Double-Angle Formulas

sin(2A) = 2sinAcosA
cos(2A) = cos2A - sin2A
cos(2A) = 2cos2A - 1
cos(2A) = 1 - 2sin2A
Example

Prove the identity: (sec2x - 1)cot2x = 1.

  1. Left Side:
    (sec2x - 1)cot2x
  2. Substitute using Pythagorean Identity:
    sec2x - 1 = tan2x
    = tan2x × cot2x
  3. Substitute using Reciprocal Identity:
    cot2x = 1/tan2x
    = tan2x × (1/tan2x)
  4. Simplify:
    = 1

    The identity is proven.

6. Solving Trigonometric Equations

Concept

Solving a trigonometric equation involves finding the specific values of the variable that make the equation true. Solutions can be conditional, within a specific interval like 0 ≤ x < 2π, or in a general form that includes all possible solutions.

Process / Strategy
  1. Use algebraic techniques to isolate the trigonometric function.
  2. Use inverse trigonometric functions or knowledge of special angles to find a reference angle.
  3. Determine the quadrants where solutions lie based on the sign, positive or negative, of the function.
  4. Find all solutions within the given interval and, if required, write the general solution by adding integer multiples of the function's period.
Example

Solve 2sinx - 1 = 0 for 0 ≤ x < 2π.

  1. Isolate:
    2sinx = 1
    sinx = 1/2
  2. Reference Angle: From special angles, the reference angle is:
    x = π/6
  3. Quadrants: Since sinx is positive, solutions are in Quadrants I and II.
    • Quadrant I solution:
      x = π/6
    • Quadrant II solution:
      x = π - π/6 = 5π/6
  4. Final Solutions:
    x = π/6, 5π/6

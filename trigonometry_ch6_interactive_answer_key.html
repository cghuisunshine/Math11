Solve the following triangle.
Find every missing side and angle.

Given right triangle ABC with ∠C = 90°, BC = 54, and CA = 36. Find AB, ∠A, and ∠B.

AB = √(36² + 54²) = √4212 = 64.90 tan A = 54/36, so A = tan⁻¹(54/36) = 56.31° B = 90° − 56.31° = 33.69° Answer: AB = 64.90, ∠A = 56.31°, ∠B = 33.69°, ∠C = 90°.
2a
Solve triangle ABC using the Sine Law and Cosine Law.

∆ABC, where ∠B = 50°, AC = 45, and AB = 38.9.

Use the Sine Law. sin C / 38.9 = sin 50° / 45 sin C = (38.9 sin 50°) / 45 C = 41.47° A = 180° − 50° − 41.47° = 88.53° BC / sin 88.53° = 45 / sin 50° BC = 58.72 Answer: ∠A = 88.53°, ∠B = 50.00°, ∠C = 41.47°, BC = 58.72.
2b
Solve triangle ABC using side lengths.

∆ABC, AB = 27.4, BC = 19.8, and AC = 11.4.

Let a = BC = 19.8, b = AC = 11.4, c = AB = 27.4. cos A = (b² + c² − a²) / (2bc) A = 38.53° cos B = (a² + c² − b²) / (2ac) B = 21.02° C = 180° − 38.53° − 21.02° = 120.45° Answer: ∠A = 38.53°, ∠B = 21.02°, ∠C = 120.45°.
3a
Determine the number of unique triangles.

For ∆DEF, ∠F = 55°, DE = 66, and DF = 78.

This is an SSA ambiguous case. sin E / 78 = sin 55° / 66 sin E = (78 sin 55°) / 66 E ≈ 75.49° A second possible angle is 180° − 75.49° = 104.51°. Both possibilities leave a positive third angle, so both triangles are possible. Answer: 2 unique triangles.

4
Hot air balloon word problem.

Three people are watching a hot air balloon travel over their town. A, B, and C are people, and D is the balloon. Person C is 392 m directly below the balloon. Person B is 225 m away from Person C. The angle between Person A, the balloon, and Person B is 26°. How far is Person A from the hot air balloon?

First find BD: BD = √(225² + 392²) = 451.98 m Find angle CBD: ∠CBD = tan⁻¹(392/225) = 60.15° Since A, B, and C are on a straight line: ∠ABD = 180° − 60.15° = 119.85° ∠DAB = 180° − 119.85° − 26° = 34.15° Use the Sine Law: AD / sin 119.85° = BD / sin 34.15° AD = (451.98 sin 119.85°) / sin 34.15° = 698.39 Answer: Person A is 698.39 m from the hot air balloon.

Exact value of sin 300°.

Explain with words and math symbols how you can use a special triangle or the unit circle to solve for the exact value of sin 300° in standard position. Find the exact value of sin 300°.

300° is in Quadrant IV. Reference angle = 360° − 300° = 60°. In Quadrant IV, sine is negative. sin 300° = −sin 60° = −√3/2 Answer: sin 300° = −√3/2.
7
Exact trigonometric ratios from a point.

Suppose there is a coordinate (−1, √5) at the end of a terminal arm and θ represents the angle in standard position. Determine the exact values of sin θ, cos θ, and tan θ.

x = −1, y = √5 r = √(x² + y²) = √((-1)² + (√5)²) = √6 sin θ = y/r = √5/√6 = √30/6 cos θ = x/r = −1/√6 = −√6/6 tan θ = y/x = √5/(−1) = −√5 Answer: sin θ = √30/6, cos θ = −√6/6, tan θ = −√5.
8a
Solve for θ, where 0° < θ < 360°.

If tan θ = 5/4 and sin θ < 0, find θ.

tan θ = 5/4 is positive in Quadrants I and III. Since sin θ < 0, θ must be in Quadrant III. Reference angle = tan⁻¹(5/4) = 51.34° θ = 180° + 51.34° = 231.34° Answer: θ = 231.34°.
8b
Solve for θ, where 0° < θ < 360°.

If (8/3)cos θ + 7 = 5, find θ.

(8/3)cos θ + 7 = 5 (8/3)cos θ = −2 cos θ = −2 × 3/8 = −3/4 Cosine is negative in Quadrants II and III. Reference angle = cos⁻¹(3/4) = 41.41° Quadrant II: θ = 180° − 41.41° = 138.59° Quadrant III: θ = 180° + 41.41° = 221.41° Answer: θ = 138.59° or 221.41°.

1. Simplify and state where undefined

(x² - 25) / (4x² - 12x - 40)
Answer:
Factor the numerator and denominator:
x² - 25 = (x - 5)(x + 5)
4x² - 12x - 40 = 4(x² - 3x - 10) = 4(x - 5)(x + 2)

Simplify:
((x - 5)(x + 5)) / (4(x - 5)(x + 2)) = (x + 5) / (4(x + 2))

Final answer:
(x + 5) / (4(x + 2))
Undefined when x = 5 or x = -2.

2. Simplify

x / (3x² - 10x + 3) - (x - 5) / ((3x - 1)(3 - x)) - 4 / (x² + 5x - 24)
Answer:
Factor the denominators:
3x² - 10x + 3 = (3x - 1)(x - 3)
x² + 5x - 24 = (x + 8)(x - 3)
and 3 - x = -(x - 3).

So the middle fraction becomes positive after subtracting a negative form.
Lowest common denominator:
(3x - 1)(x - 3)(x + 8)

Combined numerator simplifies to:
2x² - x - 36 = (x + 4)(2x - 9)

Final answer:
((x + 4)(2x - 9)) / ((3x - 1)(x - 3)(x + 8))

3. Simplify and state where undefined

(2c / (c² - 49)) × ((c - 7) / (10c² - 2c)) ÷ (4 / (c + 7))
Answer:
Rewrite division as multiplication by the reciprocal:
(2c / (c² - 49)) × ((c - 7) / (10c² - 2c)) × ((c + 7) / 4)

Factor:
c² - 49 = (c - 7)(c + 7)
10c² - 2c = 2c(5c - 1)

Cancel common factors to get:
1 / (4(5c - 1))

Undefined from the original expression when:
c = -7, 7, 0, 1/5

4. Simplify

(x - 4) / (x + 7) - ((x² - 3x - 28) / (x² + 11x + 28)) × ((x² + 7x) / (x² - 3x))
Answer:
Factor each polynomial:
x² - 3x - 28 = (x - 7)(x + 4)
x² + 11x + 28 = (x + 7)(x + 4)
x² + 7x = x(x + 7)
x² - 3x = x(x - 3)

The product simplifies to (x - 7) / (x - 3).
So now compute:
(x - 4)/(x + 7) - (x - 7)/(x - 3)

Combine over a common denominator (x + 7)(x - 3).
The numerator becomes 61 - 7x.

Final answer:
(61 - 7x) / ((x + 7)(x - 3))

6 / (x + 5) = 1 / (3 - x)
Answer:
Cross multiply:
6(3 - x) = x + 5
18 - 6x = x + 5
13 = 7x

Final answer:
x = 13/7

(4 - x)/(x + 7) - 2x²/(x² + 5x - 14) = 5/(x - 2)
Answer:
Factor x² + 5x - 14 = (x + 7)(x - 2).
Multiply through by (x + 7)(x - 2):
(4 - x)(x - 2) - 2x² = 5(x + 7)
-x² + 6x - 8 - 2x² = 5x + 35
-3x² + x - 43 = 0
3x² - x + 43 = 0

Using the quadratic formula:
x = (1 ± √(-515)) / 6 = (1 ± i√515) / 6

6. Sum and reciprocal word problem

The sum of two numbers is 34. The sum of their reciprocals is 1/3. Determine the two numbers. Answer with a sentence.
Answer:
Let the numbers be a and b.
a + b = 34
1/a + 1/b = 1/3
(a + b) / ab = 1/3
34 / ab = 1/3
ab = 102

So the numbers are roots of:
t² - 34t + 102 = 0
t = (34 ± √748) / 2 = 17 ± √187

Final answer:
The two numbers are 17 + √187 and 17 - √187.

7. Kayakers word problem

Two kayakers paddle 14 km downstream in the same time it takes to go 6 km upstream. The current is 3 km/h. Find the rate of the kayakers in still water.
Part (a) Table:
Trip Distance (km) Rate (km/h) Time (h)
Downstream 14 r + 3 14 / (r + 3)
Upstream 6 r - 3 6 / (r - 3)

Part (b) Equation and Solution:
Same time means:
14 / (r + 3) = 6 / (r - 3)
Cross multiply:
14(r - 3) = 6(r + 3)
14r - 42 = 6r + 18
8r = 60
r = 7.5

Final answer:
The rate of the kayakers in still water is 7.5 km/h.

Bonus Question

Simplify:
((2/m) - (2/(3m + 4))) / ((2/m²) + (1/(3m + 4)))
Answer:
Simplify the numerator:
2/m - 2/(3m + 4) = (2(3m + 4) - 2m) / (m(3m + 4)) = 4(m + 2) / (m(3m + 4))

Simplify the denominator:
2/m² + 1/(3m + 4) = (2(3m + 4) + m²) / (m²(3m + 4)) = (m² + 6m + 8) / (m²(3m + 4))
m² + 6m + 8 = (m + 2)(m + 4)

Divide by multiplying the reciprocal and cancel common factors:
[4(m + 2) / (m(3m + 4))] × [m²(3m + 4) / ((m + 2)(m + 4))] = 4m / (m + 4)

Final answer:
4m / (m + 4)

1. State the key features of the quadratic function

y = (x - 6)2 - 4
a) Vertex
b) Domain
c) Range
d) x-intercepts
e) y-intercept
f) Axis of Symmetry
g) Congruent equation (parent function)
h) Graph the quadratic function with at least 5 points
Answer:
  • Vertex: (6, -4)
  • Domain: (-∞, ∞)
  • Range: y ≥ -4
  • x-intercepts: (4, 0) and (8, 0)
  • y-intercept: (0, 32)
  • Axis of Symmetry: x = 6
  • Parent function: y = x²
  • Five points: (4,0), (5,-3), (6,-4), (7,-3), (8,0)

2. State the key features of the quadratic function

y = -4x2 + 16x - 12
a) Vertex
b) Domain
c) Range
d) x-intercepts
e) y-intercept
f) Axis of Symmetry
g) Congruent equation (parent function)
h) Graph the quadratic function
Answer:
  • Rewrite in vertex form: y = -4(x - 2)² + 4
  • Vertex: (2, 4)
  • Domain: (-∞, ∞)
  • Range: y ≤ 4
  • x-intercepts: (1, 0) and (3, 0)
  • y-intercept: (0, -12)
  • Axis of Symmetry: x = 2
  • Parent function: y = x²
  • Sample graph points: (1,0), (1.5,3), (2,4), (2.5,3), (3,0)

3. Find the x- and y-intercepts of

y = 3(x - 11)2 + 5
Be sure to justify your answers.
Answer:
  • x-intercepts: none, because solving 0 = 3(x - 11)² + 5 gives (x - 11)² = -5/3, which has no real solution.
  • y-intercept: set x = 0
    y = 3(0 - 11)² + 5 = 3(121) + 5 = 368
    So the y-intercept is (0, 368).

4. Determine the equation of each parabola in the form

y = a(x - p)2 + q
Use the coordinates shown on the graphs.
4(a) Answer:
From the graph, the vertex is (-5, 4) and x-intercepts are (-9, 0) and (-1, 0).

Start with:
y = a(x + 5)² + 4

Substitute (-1, 0):
0 = a(4)² + 4
0 = 16a + 4
a = -1/4

Final equation:
y = -1/4(x + 5)² + 4
4(b) Answer:
From the graph, the vertex is (6, -10) and points include (5, -7) and (7, -7).

Start with:
y = a(x - 6)² - 10

Substitute (5, -7):
-7 = a(5 - 6)² - 10
-7 = a - 10
a = 3

Final equation:
y = 3(x - 6)² - 10

5. Transformation question

If the parabola y = x2 undergoes the following transformations:
reflected over the x-axis, compressed vertically by a factor of 1/3, translated 6 units left and 2 units up, what is its new equation?
Answer:
Reflect over x-axis: y = -x²
Vertical compression by factor 1/3: y = -(1/3)x²
Translate 6 left: y = -(1/3)(x + 6)²
Translate 2 up: y = -(1/3)(x + 6)² + 2

Final equation:
y = -(1/3)(x + 6)² + 2

6. Find the equation of the parabola

Vertex (9, -3) and an x-intercept of x = 10
Answer:
Use vertex form:
y = a(x - 9)² - 3

Since x = 10 is an x-intercept, the point (10, 0) is on the parabola.
Substitute:
0 = a(10 - 9)² - 3
0 = a - 3
a = 3

Final equation:
y = 3(x - 9)² - 3

7. Complete the square

y = -5x2 + 35x + 8
Answer:
y = -5x² + 35x + 8
y = -5(x² - 7x) + 8
x² - 7x = (x - 7/2)² - 49/4
y = -5[(x - 7/2)² - 49/4] + 8
y = -5(x - 7/2)² + 245/4 + 8
y = -5(x - 7/2)² + 245/4 + 32/4
y = -5(x - 7/2)² + 277/4

Final answer:
y = -5(x - 7/2)² + 277/4

8. Donut shop revenue problem

Present price: $3.00
Daily sales: 500 donuts
For every $0.20 increase, sales drop by 10 donuts.
What price gives the most revenue?
Answer:
Let x = number of price increases.

Price: 3 + 0.20x
Number sold: 500 - 10x

Revenue function:
R(x) = (3 + 0.20x)(500 - 10x)
R(x) = -2x² + 70x + 1500

Vertex:
x = -b / 2a = -70 / (2 × -2) = 17.5

Best price:
3 + 0.20(17.5) = 6.50

Price for maximum revenue: $6.50
Maximum revenue:
R(17.5) = 2112.50
$2112.50

Note: If only exact $0.20 steps are allowed, then $6.40 and $6.60 tie for the highest step-based revenue.

9. Maximum area of a rectangle

Determine the maximum area of a rectangle with perimeter 108 m. Also find the length and width.
Answer:
Let length = x, width = w.

Perimeter:
2x + 2w = 108
x + w = 54
w = 54 - x

Area:
A = xw = x(54 - x)
A = -x² + 54x

Vertex:
x = -b / 2a = -54 / (2 × -1) = 27
w = 54 - 27 = 27

Maximum area:
A = 27 × 27 = 729

Maximum area: 729 m²
Length: 27 m
Width: 27 m

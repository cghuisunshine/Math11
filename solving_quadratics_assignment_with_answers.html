1. Solve by graphing

Find the solutions of x² + x = -3x - 3 by graphing.
Clearly state the solutions as x = ?
Answer:
Recommended graphing approach:
Graph both sides as separate equations:
y = x² + x and y = -3x - 3.
On the graph, look for the points where the two graphs intersect.
The x-coordinates of those intersection points are the solutions to the equation.

Alternative method:
Move everything to one side and graph y = x² + 4x + 3.
Then read the x-intercepts of that parabola. They give the same solutions.

In either method, the graph shows solutions at x = -1 and x = -3.

Solutions:
x = -1, -3

2. Solve by factoring

Find the roots of 2x² + 15x - 27 = 0 by factoring.
Answer:
Find two numbers that multiply to 2(-27) = -54 and add to 15.
They are 18 and -3.

2x² + 18x - 3x - 27 = 0
2x(x + 9) - 3(x + 9) = 0
(x + 9)(2x - 3) = 0

Roots:
x = -9 or x = 3/2

3. Solve by completing the square

3(a)
-3x² + 18x - 9 = 0
Answer:
Divide by -3:
x² - 6x + 3 = 0
x² - 6x = -3
x² - 6x + 9 = -3 + 9
(x - 3)² = 6
x - 3 = ±√6

Final answer:
x = 3 ± √6
3(b)
2x² + 5x + 7/2 = 0
Answer:
Divide by 2:
x² + 5/2 x + 7/4 = 0
x² + 5/2 x = -7/4
Half of 5/2 is 5/4, and (5/4)² = 25/16.
x² + 5/2 x + 25/16 = -7/4 + 25/16
(x + 5/4)² = -3/16
x + 5/4 = ± i√3 / 4

Exact solutions:
x = (-5 ± i√3) / 4
No real roots.

4. Solve using the quadratic formula

4(a)
5x² + 2x - 7 = 0
Answer:
a = 5, b = 2, c = -7
x = (-2 ± √(2² - 4·5·(-7))) / (2·5)
x = (-2 ± √144) / 10
x = (-2 ± 12) / 10

Final answer:
x = 1 or x = -7/5
4(b)
-2x² + 3x + 8 = 9x + 5
Answer:
Move all terms to one side:
-2x² - 6x + 3 = 0
Multiply by -1:
2x² + 6x - 3 = 0
a = 2, b = 6, c = -3
x = (-6 ± √(36 + 24)) / 4
x = (-6 ± √60) / 4
x = (-6 ± 2√15) / 4

Final answer:
x = (-3 ± √15) / 2

5. Stone thrown from a cliff

A person stands on a cliff that is 55 m high. The height is modeled by:
h = 55 + 30t - 5t²
When will the stone hit the water? Round to the nearest tenth of a second.
Answer:
The stone hits the water when h = 0.
55 + 30t - 5t² = 0
Divide by -5:
t² - 6t - 11 = 0
Use quadratic formula:
t = (6 ± √(36 + 44)) / 2
t = (6 ± √80) / 2
t = (6 ± 4√5) / 2
t = 3 ± 2√5

Approximate values:
t ≈ 7.472 or t ≈ -1.472
Reject the negative time.

Final answer:
t ≈ 7.5 seconds

6. Discriminant question

State the discriminant of 3x² - 8x + k, and state for which values of k there are two different real solutions.
Answer:
For ax² + bx + c, the discriminant is Δ = b² - 4ac.
Here, a = 3, b = -8, c = k.

Δ = (-8)² - 4(3)(k)
Δ = 64 - 12k

For two different real solutions, we need Δ > 0.
64 - 12k > 0
k < 16/3

Final answer:
Δ = 64 - 12k
Two different real solutions when k < 16/3

7. Solve the inequalities and give interval notation

7(a)
2x - 9 > -3
Answer:
2x - 9 > -3
2x > 6
x > 3

Interval notation:
(3, ∞)
Number line: open circle at 3, shade to the right.
7(b)
2x² + 5 ≤ -7x
Answer:
Move all terms to one side:
2x² + 7x + 5 ≤ 0
Factor:
(2x + 5)(x + 1) ≤ 0
Critical points: x = -5/2 and x = -1

The solution is between the roots, including both endpoints.

Final answer:
-5/2 ≤ x ≤ -1
Interval notation:
[-5/2, -1]
Number line: closed circles at -5/2 and -1, shade between them.
